Work pools & workers

Decouple what runs from where it runs. Local process pool, copy-paste ready.

The three pieces

PieceRoleLives
DeploymentFlow + schedule + parameters, pointed at a poolOn the server
Work poolTyped queue scheduled runs land inOn the server
WorkerPolls a pool, executes the runsA process you run

deploy() ≠ run deploy() only registers. A triggered run sits Scheduled until a worker on that pool picks it up. No worker → nothing executes.

End-to-end (process pool, all local)

# 1. Create the pool (once)
uv run prefect work-pool create local-process --type process

# 2. Deploy from local code
#    (in 08_workers.py)
worker_pipeline.from_source(
    source=str(Path(__file__).parent),
    entrypoint="08_workers.py:worker_pipeline",
).deploy(name="worker-pipeline", work_pool_name="local-process", cron="0 8 * * *")
uv run python 08_workers.py        # registers, then exits

# 3. Start a worker (separate terminal, leave running)
uv run prefect worker start --pool local-process

# 4. Trigger a run (or let cron do it)
uv run prefect deployment run 'worker-pipeline/worker-pipeline'

Work pool CLI

CommandUse
work-pool create <n> --type processCreate a pool. Types: process (local), docker, kubernetes, …
work-pool lsList pools.
work-pool inspect <n>Pool config + default job variables.
work-pool set-concurrency-limit <n> <N>Cap simultaneous runs across the pool.
worker start --pool <n>Run a worker against the pool. Run several for more throughput.

deploy() vs serve()

serve()work pool + worker
ProcessesOne does everythingServer schedules · worker(s) execute
Survives closed terminalNoYes
Many deployments / scaleAwkwardNatural
Code locationImplicit (same process)Explicit via from_source()
Best forDev, one flowToward production

Gotchas

Things that bite
  • No worker running → runs stay Scheduled. Start worker start --pool ….
  • Worker can't find code → set from_source(source=…, entrypoint="file.py:flow"). For a local process pool, point it at the project dir.
  • Wrong pool name → deployment and worker must name the same pool.
